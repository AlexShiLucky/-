Documentation: Nios II Processor

These handbooks serve as the primary documentation for the Nios® II processor:

  • The Nios II (Gen2) Processor Reference Handbook (PDF) (ver 2015.04.02, April 2015)  answers the question "What is the Nios II Gen2 processor?" and is the primary reference for the Nios II processor architecture.
  • The Nios II (Gen2) Software Developers Handbook (PDF) (ver 2015.05.19, May 2015)  answers the question  "How do I write programs for the Nios II Gen2 processor?" and is the primary reference for programming the Nios II processor.
  • The Embedded Design Handbook (PDF) (ver 2015.12.18, Dec 2015) describes how to most effectively use the Nios II EDS tools and ihelps to increase the efficiency of developing, debugging and optimizing Nios II based embedded systems. 
  • The Embedded Peripherals IP User Guide (PDF) (ver 2015.12.16, Dec 2015)  describes peripherals provided by Altera that work seamlessly with the Nios II processor and are included with Quartus® Prime software. 
  • The Nios II Floating Point Hardware 2 Component User Guide (PDF) (ver 2015.05.28, May 2015)  describes the floating point custom instruction 2 IP module provided by Altera and included with Quartus® Prime software.


Other related documents, such as tutorials and application notes, cover specific topics that are not included in the handbooks.
